**​ChatLinker Browser Extension Privacy Policy​**​  
​**​Last Updated​**​: [2025-12-24]

​**​1. Overview​**​  
ChatLinker is a browser extension designed to aggregate and manage multiple LLM services, providing users with an immersive desktop-like interface for AI agent control. Unlike traditional bookmarks, ChatLinker offers a graphical interface and categorization system to organize your AI tools efficiently.

​**​隐私政策​**​  
​**​最近更新​**​：[2025-12-24]

​**​1. 概述​**​  
ChatLinker是一款浏览器扩展工具，旨在聚合管理多平台LLM服务，为用户提供类桌面系统的沉浸式智能体操控体验。与普通书签不同，ChatLinker通过图形化界面和分类系统高效管理您的AI工具集。

---

​**​2. Data Collection​**​  
​**​We Do NOT Collect Any Information​**​

- ChatLinker does ​**​not​**​ collect, store, or transmit any user data, including but not limited to:
    - Personal information
    - Browsing history
    - Conversations with AI services
    - Extension usage patterns

​**​2. 数据收集​**​  
​**​我们不收集任何信息​**​

- ChatLinker ​**​不会​**​收集、存储或传输任何用户数据，包括但不限于：
    - 个人信息
    - 浏览历史
    - 与AI助手的对话内容
    - 扩展使用行为

---

​**​3. Permissions​**​  
The extension requires only the following permissions:

- ​**​Network Access​**​: To connect with third-party LLM services (e.g. OpenAI, Claude).
- ​**​Storage​**​: To locally save your preferences (e.g. icon arrangement, category settings).

​**​3. 权限说明​**​  
本扩展仅需以下权限：

- ​**​网络权限​**​：用于连接第三方LLM服务（如OpenAI、Claude）。
- ​**​存储权限​**​：用于本地保存用户偏好设置（如图标排列、分类系统）。
